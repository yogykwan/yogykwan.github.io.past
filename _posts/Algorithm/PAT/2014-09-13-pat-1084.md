> 模拟，字符串

###题意
给出两串，输出第一串中包含、第二串没有的字符

###细节
- 字母转为大写

### 代码
```
#include<cstdio>
#include<iostream>
#include<algorithm>
#include<cstring>
#include<vector>
#include<cmath>
using namespace std;
int vis[100];
char s[100],b[100];
int main(){
    freopen("d.in","r",stdin);
    scanf("%s%s",s,b);
    int i,j;
    for(i=0;s[i];++i) s[i]=toupper(s[i]);
    for(i=0;b[i];++i) b[i]=toupper(b[i]);
    for(i=j=0;s[i];++i){
        if(b[j]!=s[i]){
            if(!vis[s[i]]){
                vis[s[i]]=1;
                printf("%c",s[i]);
            }
        }else ++j;
    }
    return 0;
}

```
